StackSearch - Apache Solr Demo - v0.1

Tweet Follow @_saumitra_

Demo to show out of the box capabilities of Apache Solr. Motivation behind this application is to provide a configurable UI and middleware which can be used for quick POCs for any type of dataset.

In this application we have indexed data from stackexchange sites using dumps available at https://archive.org/details/stackexchange.

This demo shows following Solr features:

  • Full Text Search, Highlighting
  • Faceting, Pagination, Sorting
  • Clustering, More Like this, AutoComplete
  • Range and function queries, fuzzy search, proximity search, boosting, range faceting

What's next - v0.2

  • Completely config driven. Plugin any kind of dataset.
  • Optional middleware using Play framework
  • Query debug console
  • Multiple autocomplete suggesters
  • Opensource UI, middleware and backend(indexer)
